动态规划特点：

* 最优解
* 整体最优解依赖各个子问题最优解
* 把大问题拆解成各个子问题，子问题继续拆解
* 从上往下分析问题，从下往上解决问题

解决思路

* 确定从上到下的递归公式
* 初始化值
* 遍历可能性，填充子问题的解

```
void dynamicPlan(int input) {
    if 判断最开始情况 {
        return xxx;
    }
    if 超出一定边界 {
        return xxx;
    }

    定义子问题存储空间，通常一维或者二维数组，长度n+1

    遍历，填充子问题的解 {
        递归公式赋值
    }

    return 最后一个问题
}
```
